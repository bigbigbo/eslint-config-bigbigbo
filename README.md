# eslint-config-bigbigbo
A eslint configuration that base on `eslint-config-airbnb-base` and `eslint-config-airbnb`.

## Install
```js
npm install eslint-config-bigbigbo --save-dev
// or
yarn add eslint-config-bigbigbo -D
```

## Usage
```js
// .eslintrc
{
  extends: "bigbigbo"
}
```
